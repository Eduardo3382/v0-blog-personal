# Guía de Enlaces Directos (Secciones)

Para facilitar la navegación y poder compartir enlaces que lleven directamente a una parte específica de la página principal, he configurado los siguientes "Anclas" (Anchor Links).

## Tabla de Enlaces Rápidos

| Sección                     | ID / Enlace (Ancla) | Descripción                                                               |
| :-------------------------- | :------------------ | :------------------------------------------------------------------------ |
| **Inicio**                  | `#`                  | Parte superior de la página (Hero).                                       |
| **Último Artículo**         | `#post`              | El artículo más reciente destacado con imagen grande.                     |
| **Próximos Talleres**       | `#proximos`          | Sección de cursos o talleres que se dictarán pronto.                      |
| **Último Boletín**          | `#boletin`           | Sección con el link al último boletín de "Pastillitas".                   |
| **Sobre Mí**                | `#about`             | Biografía y pilares fundamentales.                                        |
| **Artículos Anteriores**    | `#blog`              | Lista de los posts realizados anteriormente.                              |
| **Galería de Fotos**        | `#fotos`             | Vista previa de las últimas imágenes.                                     |
| **Enseñanza / Recursos**    | `#ensenanza`         | Métodos de enseñanza, clases y recursos gratuitos.                        |
| **Contacto**                | `#contact`           | Redes sociales y formulario de contacto.                                  |

## Cómo usarlos

1. **Uso en Navegador:** Puedes añadir el ID al final de tu URL. Ejemplo: `https://eduardo.net.ar/#boletin`.
2. **Uso en Redes Sociales:** Cuando compartas un boletín o un artículo nuevo, puedes usar estos links para que la gente no se pierda buscando en la página.

---
> [!NOTE]
> Estos enlaces ya están integrados en el código de los componentes. Si en el futuro necesitas cambiar un ID, deberás hacerlo en el archivo `.tsx` correspondiente de la carpeta `components/`.
